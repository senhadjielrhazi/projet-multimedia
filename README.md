# Projet Multimedia
============
Ce projet nomme Multimedia s'est tenu sur la duree d'un semestre avec un autre etudiant et consiste a modeliser les deplacements de robots des pions sur un damier.

Programme : 
- Langage C sur UNIX
- Algorithme de déplacement
- Reconstruction graphique


Projet Multimedia ENPC 2002, Paris France, [Presentation PowerPoint](https://www.academia.edu/33725645/Bras_de_robot_sur_damier_OPENGL), O. Senhadji El Rhazi.
